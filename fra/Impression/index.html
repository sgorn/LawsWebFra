Impression

Version PDF

Pour imprimer un document en version PDF, simplement cliquer sur le lien fourni à cet effet. Si le format de tableaux, formulaires ou représentations graphiques est incorrect, veuillez nous contacter.

Version HTML

On peut imprimer n’importe quelle page du Site Web de la législation (Justice) en utilisant l’option d’impression disponible sur le navigateur internet utilisé. Le site Web supprime automatiquement les menus et les parties spécifiques au site Web dans le but d’imprimer une copie papier en format HTML.

Renseignements divers

Cliquer sur « Aperçu avant impression... », à partir du menu « Fichier », permet de voir si la mise en page est appropriée. Si le texte dépasse le cadre de la page, sélectionner l’orientation « paysage » à partir du menu « Mise en page ». Le menu de « Mise en page... » de la version 1.5 (ou plus) de Mozilla Firefox offre l’option « Adapter à la taille de la page ».

Sur ce site, le ministère de la Justice du Canada a utilisé des feuilles de style en cascade (FSC) pour améliorer l’accès aux lois et aux règlements codifiés du Canada. Bien que l’utilisation des FSC soit le meilleur moyen de créer des sites Web entièrement accessibles, les usagers qui utilisent d’anciens fureteurs ou d’anciennes imprimantes peuvent avoir des difficultés d’impression.

Pour régler les problèmes liés à l’impression, il est recommandé de mettre le fureteur à jour. Internet Explorer 7 (ou plus) et Mozilla Firefox 1.5 (ou plus) sont recommandés.
